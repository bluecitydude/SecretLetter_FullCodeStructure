Overview

The SecretLetter API allows you to programmatically encrypt and decrypt messages using our multi-layer encryption algorithm. The API is stateless and does not store any of your data.

Endpoint Details

POST https://secretletter-api.onrender.com/Backend/Controller/Controller.php

Content-Type: application/json
Response-Type: json

Payload Structure

Encryption Mode

{
  "mode": "encrypt",
  "message": "Your secret message",
  "passphrase": "YourPassword123",
  "times": "1" // 1 to 5
}

Decryption Mode

{
  "mode": "decrypt",
  "secret": "EncryptedStringHere==",
  "passphrase": "YourPassword123",
  "times": "1" // Must match encryption
}

Response Format

JSON 
{
  "status": "success", // or "error"
  "response": "The resulting string or error message"
}

Integration Examples

JavaScript (Fetch) 
const encryptMessage = async () => {
  const payload = {
    mode: 'encrypt',
    message: 'Hello World',
    passphrase: 'MySecretPassword',
    times: '3'
  };

  const response = await fetch('https://secretletter-api.onrender.com/Backend/Controller/Controller.php', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(payload)
  });

  const data = await response.json();
  console.log(data);
};
Python (requests) 
import requests

url = 'https://secretletter-api.onrender.com/Backend/Controller/Controller.php'
payload = {
    'mode': 'decrypt',
    'secret': 'U2FsdGVkX1...',
    'passphrase': 'MySecretPassword',
    'times': '3'
}

response = requests.post(url, json=payload)
print(response.json())
PHP (cURL) 
<?php
$url = 'https://secretletter-api.onrender.com/Backend/Controller/Controller.php';
$data = array(
    'mode' => 'encrypt',
    'message' => 'Hello World',
    'passphrase' => 'MySecretPassword',
    'times' => '3'
);

$ch = curl_init($url);
$payload = json_encode($data);
curl_setopt($ch, CURLOPT_POSTFIELDS, $payload);
curl_setopt($ch, CURLOPT_HTTPHEADER, array('Content-Type:application/json'));
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
$result = curl_exec($ch);
curl_close($ch);

echo $result;
?>
